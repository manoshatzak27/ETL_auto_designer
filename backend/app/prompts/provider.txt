## Deduplication

Deduplicate by `provider_source_value` when configured. If multiple source rows have the same `provider_source_value`, emit only one PROVIDER row (first occurrence). If `provider_source_value_col` is not configured, emit one row per source row.

## care_site_id lookup

Load `ETL_OUTPUT_DIR/care_site.csv` if it exists. Build a dict mapping `care_site_source_value → care_site_id`. For each provider row, look up the value of `care_site_source_value_col` in this dict. If no match or file absent, set `care_site_id` to None.
